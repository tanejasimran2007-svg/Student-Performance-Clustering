# Student Performance Analysis using K-Means Clustering

## Objective

The objective of this project is to identify groups of students with similar academic performance patterns using K-Means clustering.

---

## Dataset

The dataset contains 1000 student records with the following features:

- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

---

## Techniques Used

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- StandardScaler
- K-Means Clustering
- Elbow Method
- Silhouette Score
- Cluster Analysis

---

## Key Findings

- Three clusters of students were identified:
  - High Performing Students
  - Average Performing Students
  - Low Performing Students

- Reading and writing scores showed the strongest correlation.

- Students completing test preparation courses appeared more frequently in the high-performing cluster.

- Students in the high-performing cluster were more likely to have standard lunch.

---

## Results

Silhouette Score: **0.406**

---

## Limitations

- Only academic scores were used for clustering.
- K-Means assumes spherical clusters.
- The dataset contains only 1000 student records.
